# 🏹 Compound Archery — Train Smart

Application mobile-first pour archers compound.

## Fonctionnalités

- **Entraînement** — 4 séances (18m indoor + 50m outdoor) avec checklist interactive, fiches techniques et liens vidéo YouTube
- **Viseur** — Tableau de sight marks par distance (élévation + windage) avec courbes automatiques
- **Mon Arc** — 8 réglages essentiels avec fiches d'aide FFTA intégrées
- **Glossaire** — Définitions des termes anglais et techniques au tap

## Héberger sur GitHub Pages

1. Créer un repo GitHub (ex: `archery-app`)
2. Upload `index.html` à la racine
3. Aller dans **Settings → Pages**
4. Source : `Deploy from a branch` → branche `main`, dossier `/ (root)`
5. Cliquer **Save** — l'URL sera `https://[votre-pseudo].github.io/archery-app/`

> Aucun build, aucune dépendance à installer. Un seul fichier `index.html`.
